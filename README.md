# Unique-Bortieh Financial Consult
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Premium financial consulting website built with vanilla HTML, CSS, and JavaScript. Modern, responsive design showcasing services, about, testimonials, and contact form.

## 🚀 Features
- **Hero Section**: Fullscreen finance bg image with animated stats & charts
- **Responsive Design**: Mobile-first, works on all devices
- **Smooth Animations**: Scroll reveals, counters, parallax, hover effects
- **Interactive Elements**: Navbar, form validation, back-to-top, ticker
- **Sections**: About, Services, Solutions, Features, Stats, Testimonials, CTA, Contact
- **Performance**: Optimized CSS vars, no frameworks, ~200KB total

## 📱 Sections Overview
| Section | Visuals |
|---------|---------|
| Hero | finance1.jpg bg + charts |
| About | finance3.jpg portrait |
| Services | 6 unique finance bgs (finance1-4, finace5/6, fincae7) w/ overlays |
| Solutions/Features/Stats | Icons & gradients |
| Testimonials | Styled avatars |

## 🎨 Design
- **Colors**: Green theme (#4CAF50) on dark/light
- **Fonts**: Playfair Display (headings), DM Sans (body), DM Mono (mono)
- **Icons**: Phosphor Icons CDN
- **Effects**: Glassmorphism, orbs, gradients, shadows

3. Responsive preview: Resize window or DevTools


## 📱 Responsive Breakpoints
- Desktop: >992px
- Tablet: 768px
- Mobile: <480px

## 🔧 Customization
- **Colors**: Edit CSS vars in `:root`
- **Images**: Swap JPGs in `images/`, update `data-service-img`/`src`
- **Content**: Edit sections in `index.html`
- **Icons**: Phosphor Icons (unpkg CDN)

## 🛠 Technologies
- HTML5 Semantic
- CSS3 (Grid, Flexbox, Custom Props, Animations)
- Vanilla JavaScript (no frameworks)
- Phosphor Icons
